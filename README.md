# Applied Data Engineering

Welcome to the **Applied Data Engineering** repository! This repository serves as a centralized hub for my hands-on practice, experiments, and projects related to various Data Engineering and Big Data technologies.

## 🎯 Purpose

The primary goal of this repository is to demonstrate practical proficiency in designing, building, and managing data pipelines, infrastructure, and analytical solutions. It contains code, configurations, and documentation reflecting real-world data engineering scenarios.

## 🛠️ Technologies & Tools

This repository includes projects and exercises utilizing the following technologies:

*   **Containerization & Orchestration:** Docker, Docker Compose
*   **Big Data Processing:** Apache Spark
*   **Workflow Orchestration:** Apache Airflow
*   **Databases & Data Warehousing:** SQL, Snowflake
*   **Data Transformation:** dbt (Data Build Tool)
*   *(More technologies will be added as the repository grows)*

## 📂 Structure

*(Directory structure will be updated as new projects are added. Examples include:)*

*   `/docker` - Dockerfiles and Compose files for local environments.
    *   `/docker/airflow` - Airflow local cluster setup.
    *   `/docker/airflow/dbt_project` - dbt models mounted natively into Airflow operators.
*   `/spark` - Data processing and transformation scripts.
*   `/airflow` - Directed Acyclic Graphs (DAGs) for scheduling and orchestration.
*   `/sql` - Advanced querying, schema design, and DDL/DML scripts.
*   `/snowflake` - Cloud data warehousing setups and transformations.

## 🚀 Getting Started

Detailed instructions for running individual projects or modules will be provided in local `README.md` files within their respective directories. Ensure you have the necessary prerequisites (such as Docker, Python, or relevant database clients) installed before executing the code.
